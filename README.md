#### 🧠 Java-NLP — Natural Language Processing Toolkit
> A production-ready Java-based NLP engine built with Stanford CoreNLP. Designed for enterprise integration with a modular, extensible architecture.

**What it does:**
- 🔤 **Tokenization** — splits raw text into meaningful tokens
- 📝 **Lemmatization** — reduces words to their base/root forms
- 🏷️ **POS Tagging** — labels each word as noun, verb, adjective, etc.
- 🏢 **Named Entity Recognition (NER)** — identifies persons, organizations, locations
- 💬 **Sentiment Analysis** — classifies text as positive, negative, or neutral
- 🔌 **Modular design** — plug into any enterprise Java application easily

`Java` `Stanford CoreNLP` `NLP` `NER` `Sentiment Analysis` `Maven`

---

#### 🏷️ NER Spring Boot — Named Entity Recognition API
> A Spring Boot microservice that wraps Stanford NLP's NER capabilities behind a clean REST API — making NLP accessible to any service in your architecture.

**What it does:**
- 📡 Exposes `POST /api/ner` endpoint — send text, get structured entities back
- 🏢 Extracts **PERSON**, **ORGANIZATION**, **LOCATION**, **DATE**, **MONEY** entities
- 📦 Returns structured JSON response for easy downstream consumption
- ⚡ Stateless microservice — horizontally scalable

```json
Input:  "Vikrant works at Speridian Technologies in Bengaluru"
Output: { "PERSON": ["Vikrant"], "ORG": ["Speridian Technologies"], "LOCATION": ["Bengaluru"] }
```

`Spring Boot` `Stanford NLP` `REST API` `Microservice` `JSON` `NER`

---

#### 🧬 Ollama SpringBoot — GenAI API Scaffold
> A Spring Boot application scaffolded to work with local LLMs via Ollama. Designed as a production-ready foundation for GenAI-powered enterprise applications.

**What it does:**
- 🌐 Exposes `POST /api/query` — accepts natural language questions, returns AI responses
- 🔌 Integrates with **Ollama** to run LLMs locally (Llama, Mistral, etc.)
- 🏗️ Clean architecture designed for easy swap of any LLM backend
- 🔒 No external API dependency — fully self-hosted AI pipeline
- 📈 Ready for RAG (Retrieval-Augmented Generation) extensions

`Spring Boot` `Ollama` `LLM` `GenAI` `REST API` `Local AI` `Java`

#### 🎙️ Speech Recognition — Audio Transcription Engine
> A Python-based audio transcription tool that converts speech to text using Google's Web Speech API, with built-in sentiment analysis on the transcribed output.

**What it does:**
- 🎤 Processes **WAV audio files** and transcribes them to text

- 💬 Runs **sentiment analysis** on the transcribed text output
- 📁 Batch processing support for multiple audio files
- 🔧 Easily extendable to other audio formats (mp3, flac)
